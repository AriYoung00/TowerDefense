Adds health to a tower. Param health is added to _health. If the tower's current health added to the health is greater than the max health for the defined TowerType, then _health will be set to the max health for the defined TowerType.

void Towers::Tower::destroy ( )

Mark a tower for destruction and begin playing the destroy animation. Seperate from the deletion operator in order to make sure the animation finishes playing before the object itself is destroyed. Sets _shouldDestroy to true.
